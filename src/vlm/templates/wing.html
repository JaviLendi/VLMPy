Plane Definition

Plane Definition Info

  • Flight Condition:
    • Flight Velocity: Aircraft speed in meters per second (m/s). Determines aerodynamic forces.
    • Air Density: Density of air in kilograms per cubic meter (kg/m³). Affects lift and drag calculations.
    • Angle of Attack: Angle between the wing chord and the airflow (degrees). Influences lift and stall behavior.
    • Side Slip Angle: Angle between the aircraft's heading and its actual path (degrees). Important for lateral stability.
  • Wing Geometry:
    • Chord Root/Tip: Length of the wing at the root and tip (meters). Defines wing shape and area.
    • Span Fraction: Fraction of total span for each section. Used to segment the wing for analysis.
    • Sweep: Leading edge sweep angle (degrees). Affects aerodynamic characteristics and stability.
    • Dihedral: Upward angle of the wings (degrees). Contributes to roll stability.
    • NACA Root/Tip: Airfoil code at root and tip. Specifies the aerodynamic profile.
    • Flap: Control surface parameters (start/end position, hinge chord, deflection angle/type). Used for lift and control augmentation.
  • Horizontal Stabilizer:
    • X/Z Translate: Position along fuselage and vertical axis (meters). Sets stabilizer location.
    • NACA Root/Tip: Airfoil code for stabilizer sections.
    • Chord Root/Tip: Section lengths (meters).
    • Span Fraction: Fraction of total span for each section.
    • Sweep/Dihedral: Leading edge sweep and dihedral angles (degrees).
    • Alpha: Angle of attack (degrees).
  • Vertical Stabilizer:
    • X/Z Translate: Position along fuselage and vertical axis (meters).
    • NACA Root/Tip: Airfoil code for stabilizer sections.
    • Chord Root/Tip: Section lengths (meters).
    • Span Fraction: Fraction of total span for each section.
    • Sweep: Leading edge sweep angle (degrees).
    • Dihedral: Dihedral angle (degrees).
    • Alpha: Angle of attack (degrees).
  • Discretization:
    • Panels Along Span: Number of panels dividing the wing span. Higher numbers increase accuracy.
    • Panels Along Chord: Number of panels dividing the wing chord. Controls resolution of aerodynamic calculations.
  • Configuration Management:
    • Save/Load: Store and retrieve plane configurations as JSON files for repeatable analysis.

Wing Geometry

Wing Geometry Info

Use the "Add Section" button to create multiple wing sections. For each section, define:

  • Chord Root/Tip: Length of the wing section at root/tip.
  • Span Fraction: Fraction of total span for this section.
  • Sweep: Leading edge sweep angle (degrees).
  • Dihedral: Dihedral angle (degrees).
  • NACA Root/Tip: Airfoil code at root/tip.
  • Flap: Optional control surface parameters.
You can define a flap or aileron for each section by specifying its start and end positions along the chord, hinge chord fraction, deflection angle, and type (simple or plain).

Each section is defined relative to the previous one, allowing for complex wing shapes.

You can remove the last section using the "Remove Section" button.
